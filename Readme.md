# 프로젝트 소개

### 주제
영화 & 영화관 리뷰 사이트

### 목적
- 영화를 보기 위해 영화의 리뷰를 확인할 수 있다.
- 영화 뿐만 아니라 영화관에 대한 리뷰도 확인할 수 있다.
- 두 가지의 리뷰를 통해 어떤 영화관에서 어떤 영화를 볼지를 결정한다.
